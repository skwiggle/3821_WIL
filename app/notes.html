Unity command line

Command Description
Windows: "%PROGRAMFILES%\Unity\Editor\Unity.exe"
Mac OS X: /Applications/Unity/Unity.app/Contents/MacOS/Unity 		Open unity
Windows: "%PROGRAMFILES%\Unity\Editor\Unity.exe"
Mac OS X: /Applications/Unity/Unity.app/Contents/MacOS/Unity -quit -batchmode -serial SB-XXXX-XXXX-XXXX-XXXX-XXXX -username 'JoeBloggs@example.com' -password 'MyPassw0rd' 		Opens unity silently
Windows: "%PROGRAMFILES%\Unity\Editor\Unity.exe" -quit -batchmode -returnlicense
Mac OS X: /Applications/Unity/Unity.app/Contents/MacOS/Unity -quit -batchmode -returnlicense 		Returns license back to license server
Windows:
  • "%PROGRAMFILES%\Unity\Editor\Unity.exe" -batchmode -createManualActivationFile -logfile
  • "%PROGRAMFILES%\Unity\Editor\Unity.exe" -batchmode -manualLicenseFile (yourulffile) -logfile
Mac OS X:
  • /Applications/Unity/Unity.app/Contents/MacOS/Unity -batchmode -createManualActivationFile -logfile
  • /Applications/Unity/Unity.app/Contents/MacOS/Unity -batchmode -manualLicenseFile (yourulffile) -logfile
 Creates a license file and imports it into unity hub
